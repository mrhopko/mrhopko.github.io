Adding Sparklines First you need to first install.packages(“sparkline”). DTHelper helps you embed sparklines into you datatables. Example: library(data.table) library(DT) library(DTHelper) library(sparkline)

create a data set using data.table

x <- data.table(week_no = c(rep(c(1:10),3)), col1 = c(1:30), col2 = c(21:50), col3 = round(runif(30, min = 0, max = 30),0), grp = c(rep("a",10), rep("b",10), rep("c",10))) head(x)

week_no col1 col2 col3 grp

1: 1 1 21 7 a

2: 2 2 22 28 a

3: 3 3 23 1 a

4: 4 4 24 7 a

5: 5 5 25 8 a

6: 6 6 26 12 a

A cell in a table usually has 1 value, a sparkline requires a sequence of values.

R Getting More From datatables DT

Introduction The DT package is an R package built on top of the javascript library datatables by the RStudio team. They can be used in RMarkdown and Shiny to create interactive tables. If you want to learn the basics, there are plenty of tutorials. The DT R package allows us to write R code to generate the HTML widgets. The package manages to surface much of the functionality of the javascript library and, with a little additional javascript, can be extended further.
