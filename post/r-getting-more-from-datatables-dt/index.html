R Getting More From datatables DT

Thu, Apr 20, 2017 R, Data Visualisation R, datatables, DT, R Markdown, crosstalk

Introduction

The DT package is an R package built on top of the javascript library datatables by the RStudio team. They can be used in RMarkdown and Shiny to create interactive tables. If you want to learn the basics, there are plenty of tutorials. The DT R package allows us to write R code to generate the HTML widgets. The package manages to surface much of the functionality of the javascript library and, with a little additional javascript, can be extended further. From a data visualisation perspective, this flexibility creates an opportunity. We can weave a striking amount of information into a single table while keeping it neat and readable.
